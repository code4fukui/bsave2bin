# bsave2bin

MSXのBSAVE形式のファイルから7バイトのヘッダーを取り除き、純粋なバイナリデータファイルに変換します。

## 機能

- MSXのBSAVEファイルからバイナリデータを抽出します。
- シンプルなウェブインターフェースとコマンドラインツールの両方を提供します。
- ウェブアプリでは、BSAVEのヘッダー情報（開始アドレス、終了アドレス、実行アドレス）を表示します。

## 使い方

このツールは、ウェブアプリまたはコマンドラインインターフェースの2通りの方法で使用できます。

### ウェブアプリ

**デモ: [bsave2bin](https://code4fukui.github.io/bsave2bin/)**

ウェブアプリは、レトロなMSX風の使いやすいインターフェースを提供します。BSAVEファイルをページにドラッグ＆ドロップするか、ファイルを選択してください。

- ファイルのメタデータ（開始アドレス、終了アドレス、実行アドレス）が表示されます。
- 変換されたバイナリファイルが自動的にダウンロードされます。

### コマンドラインツール

CLIツールは、ローカルで変換を行うためのDenoスクリプトです。

#### 必要条件

- [Deno](https://deno.land/) ランタイム

#### コマンド

URLから直接スクリプトを実行し、入力するBSAVEファイルと出力するバイナリファイル名を指定してください。

```sh
deno run -A https://code4fukui.github.io/bsave2bin/bsave2bin.js [bsave file] [bin file]
```
